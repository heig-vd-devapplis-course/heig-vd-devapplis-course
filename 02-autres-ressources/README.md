## Avant le début du cours

### Message envoyé aux personnes qui étudient avant le premier cours

```text
[DévAppliS 2025-2026] Informations générales sur le cours

Bonjour tout le monde,

Nous espérons que vous allez bien.

Noemi et moi-même serons vos enseignants pour le cours "Développer une application web simple (DévAppliS)". Nous nous réjouissons de vous retrouver la semaine prochaine **en R102, à 13:00** pour vivre cette expérience ensemble.

Nous commencerons avec une introduction du cours avec le mandant externe qui sera présent pour vous présenter le projet que vous allez réaliser durant la semaine.

Vous trouverez toutes les informations sur le cours dans le dépôt GitHub suivant : https://github.com/heig-vd-devapplis-course/heig-vd-devapplis-course.

**Nous vous demandons de prendre connaissance du README principal de ce dépôt GitHub avant le début du cours.** Le contenu n'est pas très long, mais il contient des informations importantes sur le cours, notamment les objectifs, les formations des groupes, les modalités d'évaluation et les ressources disponibles.

Pour les personnes qui n'auraient pas suivi le cours "ProgServ1" cette année avec moi, je vous demanderai de suivre le support de cours suivant de la section "Installer et configurer l'environnement de développement local" à la section "Démarrer le serveur de développement PHP" (y compris) afin de vous assurez
que vous avez un environnement de développement local fonctionnel et que vous êtes à l'aise avec les outils de développement web : https://github.com/heig-vd-progserv-course/heig-vd-progserv1-course/blob/main/01-contenus-du-cours/01.02-introduction-a-php/01-mini-projet/README.md

Je passerai vers chacun.e d'entre vous pour vérifier que vous avez un environnement de développement local fonctionnel et que vous êtes à l'aise avec les outils de développement web. N'hésitez pas à me contacter si vous avez des questions ou des problèmes.

Nous vous souhaitons une bonne journée et à bientôt,
Ludovic Delafontaine, Noemi Romano et Cédric Junillon
```
