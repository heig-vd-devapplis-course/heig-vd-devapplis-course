pour la préparation et la gestion du cours.

## Avant le début du cours

### Message envoyé aux personnes qui étudient avant le premier cours

```text
[DévAppliS 2025-2026] Informations générales sur le cours

Bonjour tout le monde,

Nous espérons que vous allez bien.

Noemi et moi-même serons vos enseignants pour le cours "Développer une application web simple (DévAppliS)". Nous nous réjouissons de vous retrouver **lundi en R102, à 13:00** pour vivre cette expérience ensemble.

Nous commencerons avec une introduction du cours avec le mandant externe qui sera présent pour vous présenter le projet sur lequel vous allez travailler durant la semaine.

Vous trouverez toutes les informations sur le cours dans le dépôt GitHub suivant : https://github.com/heig-vd-devapplis-course/heig-vd-devapplis-course.

**Nous vous demandons de prendre connaissance du README principal de ce dépôt GitHub avant le début du cours.** Le contenu n'est pas très long, mais il contient des informations importantes sur le cours, notamment les objectifs, les formations des groupes, les modalités d'évaluation et les ressources disponibles.

**Pour les personnes qui n'auraient pas suivi le cours ProgServ1 avec moi cette année**, je vous demanderai de suivre le support de cours suivant afin de vous assurer que vous ayez un environnement de développement local fonctionnel : https://github.com/heig-vd-progserv-course/heig-vd-progserv1-course/blob/main/01-contenus-du-cours/01.02-introduction-a-php/01-mini-projet/README.md. Il faut appliquer les éléments de la section "Installer et configurer l'environnement de développement local" à la section "Démarrer le serveur de développement PHP" (y compris).

Pour les autres personnes, normalement, vous avez déjà tout le nécessaire. Dans le doute, je vous invite à faire de même.

Je passerai vers chacun.e d'entre vous pour vérifier que vous avez un environnement de développement local fonctionnel.

N'hésitez pas à nous contacter si vous avez des questions ou des problèmes.

Nous vous souhaitons une bonne journée et un bon weekend en avance !
Ludovic Delafontaine et Noemi Romano
```
